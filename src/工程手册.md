# 第一章　设施概况

## 1.1 子午线站基本参数

| 项目 | 参数 |
|---|---|
| 设施名称 | 子午线深海站 |
| 英文名称 | Meridian Abyssal Station |
| 管理单位 | POU 深海设施管理局 |
| 设计驻站人数 | 4 人标准组，可短期容纳 8 人交接 |
| 标准轮换周期 | 36 个月 |
| 建议交接期 | 72 小时 |
| 标称深度 | 4,200 m 级 |
| 设计外压 | 43 MPa 级 |
| 主体结构 | 钛合金承压壳 + 复合隔热层 + 模块化内舱 |
| 能源系统 | 小型封闭式反应堆 + 蓄能阵列 |
| 通讯方式 | 低频声学链路、窄带压缩上传、月窗同步 |
| 主任务 | 深海地质、热盐层、声学异常、生态压力环境观测 |

## 1.2 建站目的与任务范围

子午线站位于热盐噪声带边缘，主要任务包括：

1. 长周期深海地质与热盐扰动监测；
2. 深海高压环境下材料稳定性测试；
3. 声学传播窗口与低频通讯实验；
4. 深海生态微样本采集与长期培养；
5. 封闭式生命保障系统可靠性验证；
6. 人员长期驻站心理与生理适应性研究。

驻站成员应理解：子午线站并非短期科考船附属设施，而是长期自主运行的海底固定站。大多数问题必须在站内解决，不能依赖即时外部支援。

## 1.3 站体布置总览

子午线站为横向主廊结构，核心舱段由中央走廊连接。标准舱段包括：

- 对接舱
- 生活舱
- 餐舱
- 医疗舱
- 通讯室
- 工程舱
- 动力舱
- 观测穹顶
- 储备舱
- 逃生舱接口

站体内部按水密区划分为 A、B、C、D 四个隔离段。任意单一区段受损时，其余区段应能维持最低生命保障不少于 96 小时。

## 1.4 环境条件与设计边界

子午线站所处区域具有以下特征：

1. 海水压力极高，外壳微形变属于正常现象；
2. 热盐层扰动会影响声学通讯质量；
3. 站体受潮汐与地层微位移影响，会产生周期性低频共振；
4. 长期封闭环境会放大睡眠障碍、焦虑、幻听感、时间感偏移等常见反应；
5. 对观测穹顶、外壳接口、对接环及动力舱承压件的检查必须严格按周期执行。

# 第二章　站体组成

## 2.1 对接舱与减压过渡区

对接舱用于深潜器停靠、人员转移、物资接收及小型外部作业准备。

**标准流程：**

1. 确认外部对接环压力平衡；
2. 等待站控显示绿色锁定；
3. 双方确认舱门编号；
4. 先开外侧过渡门，后开内侧水密门；
5. 人员进入后完成身份确认与污染物检查；
6. 物资转运后进行除湿与盐雾清洁。

**注意：**
对接舱不得作为长期储物空间。任何阻挡水密门闭合的物品都必须立即清除。

## 2.2 生活舱

生活舱包括个人宿舍、洗漱区、短时休息区及公共储物柜。

**生活舱要求：**

- 每名成员配备独立睡眠舱位；
- 床边不得堆放硬质工具、燃料、电池组；
- 私人物品须控制在个人柜容量内；
- 每周进行一次防霉、防凝露检查；
- 夜间照明默认蓝光低亮度模式，不得擅自拆改灯带。

生活舱是人员恢复体力的主要区域，非紧急情况下不得在生活舱进行高噪声维修。

## 2.3 餐舱与储备区

餐舱承担进餐、短会、营养补给发放功能。

**食品储备原则：**

1. 标准驻站组按 4 人 36 个月配置基础口粮；
2. 交接期按 8 人 7 日配置额外补给；
3. 高浓缩营养剂仅用于医疗、交接应激、作业后恢复；
4. 所有异常消耗须记录并上报；
5. 动物食品与人员食品分区存储。

餐舱同时设置热风循环口，用于局部除湿。不得覆盖或堵塞热风口。

## 2.4 医疗舱

医疗舱配置基础急救、睡眠监测、生理扫描、脑电记录与小型隔离床。

**医疗官职责包括：**

- 每周生命体征检查；
- 每月心理状态问卷；
- 睡眠异常统计；
- 缺氧、感染、压力伤害、营养失衡处理；
- 交接期间新旧成员适应性评估。

医疗舱不得被用作普通休息室或仓库。

## 2.5 通讯室

通讯室用于对外通讯、月报上传、站内广播、数据压缩与离线缓存。

**通讯室标准配置：**

- 主通讯控制台；
- 月窗同步终端；
- 离线缓存阵列；
- 物理权限卡槽；
- 声学链路状态屏；
- 单人值守座椅；
- 独立应急供电单元。

通讯员应保证通讯室清洁、干燥、无非授权存储设备。

## 2.6 工程舱

工程舱是站体维护、结构监测和系统诊断中心。

**主要设备：**

- 站体结构投影台；
- 外壳应变监测终端；
- 生命保障系统副控；
- 热管理控制台；
- 水循环状态柜；
- 工具墙与备件柜；
- 小型焊接及密封修补组件。

工程舱内工具必须按编号归位。缺失工具视为安全事件。

## 2.7 动力舱

动力舱包括主反应堆、热交换系统、蓄能阵列、配电母线与安全联锁装置。

**进入要求：**

1. 非工程师不得单独进入；
2. 进入前向站长报备；
3. 进入后开启个人剂量与温度监测；
4. 不得绕过双人确认联锁；
5. 未经授权不得执行过载、旁路或强制冷却程序。

动力舱是全站最高风险舱段之一，任何异常声响、焦味、温升或电弧痕迹必须立即记录。

## 2.8 观测穹顶

观测穹顶用于目视观察、影像采集、外部灯阵校准及长期生态记录。

**结构说明：**

观测穹顶由多层耐压透明复合材料组成，包括：

- 外承压层；
- 中间缓冲层；
- 内安全层；
- 边缘热补偿密封圈；
- 除雾与温控线路。

**使用规范：**

1. 每次进入前确认穹顶内侧无裂纹、雾化、霜点；
2. 不得用硬物敲击透明层；
3. 不得将高温灯具、焊枪、电池组靠近穹顶边缘；
4. 发现夹层湿度升高，应通知工程师；
5. 长时间观察应每 30 分钟休息 10 分钟，避免视觉疲劳与空间压迫感。

## 2.9 外部作业接口

站体外部设有机械臂接口、取样管路、灯阵维护点、声呐校准点及深潜器停泊标记。

外部作业原则上应通过深潜器或遥控机械臂完成，不建议人员直接出舱。

## 2.10 逃生舱与应急浮升单元

逃生舱用于不可恢复事故下的人员撤离。

**撤离条件包括：**

- 主体进水无法隔离；
- 供氧系统不可恢复；
- 动力系统失控；
- 火灾无法扑灭；
- 站长或两名以上高级权限人员确认站体失去维生能力。

逃生舱强制上浮前必须确认：

1. 人员全部固定；
2. 医疗包、定位信标、压缩通讯包已装载；
3. 舱门闭锁；
4. 外部释放机构绿灯；
5. 浮升路径无对接干涉。

# 第三章　岗位职责

## 3.1 站长

站长是子午线站现场最高负责人。

**主要职责：**

- 负责驻站任务执行；
- 维护人员纪律与安全；
- 审批外部作业、深潜器使用和高风险维修；
- 主持每日简报；
- 在紧急状态下决定隔离、停机、撤离或等待救援；
- 监督轮换交接。

站长应每日检查一次站控总览，并确认生命保障、动力、通讯、库存四项指标处于可接受范围。

## 3.2 通讯员

**主要职责：**

- 维护通讯室设备；
- 编制、压缩、校验并上传月报；
- 管理离线缓存；
- 记录月窗期状态；
- 接收并分发总部指令；
- 保管通讯权限卡。

通讯员不得擅自删除站内日志。故障缓存应标记，不得覆盖。

## 3.3 工程师

**主要职责：**

- 维护动力、热管理、水循环、舱门、外壳监测系统；
- 执行每日与每周巡检；
- 管理工具与备件；
- 对异常噪声、振动、渗漏、温升进行诊断；
- 协助深潜器对接与外部取样。

工程师对任何未解释的结构变化拥有临时停用相关舱段的建议权。

## 3.4 医疗官

**主要职责：**

- 负责人员健康；
- 管理药品、急救包、隔离流程；
- 监测睡眠、心率、血氧、心理压力；
- 判断人员是否适合高风险作业；
- 定期向站长提交健康摘要。

医疗官可基于健康风险建议暂停某成员值班或外部作业。

## 3.5 物资与生活保障员

**主要职责：**

- 管理食品、水处理耗材、个人用品、动物用品；
- 记录库存；
- 维护餐舱、储备区与生活舱基础秩序；
- 检查垃圾压缩与回收系统；
- 交接期核对补给数量。

物资差异超过 3% 时，应生成异常消耗记录。

## 3.6 科学观察员 / 取样员

**主要职责：**

- 执行地质、水体、生态样本采集；
- 维护观察仪器；
- 标注样本编号；
- 保证原始数据至少双份备份；
- 向通讯员提交月报附件。

样本舱不得与餐舱、医疗舱或生活舱交叉使用。

## 3.7 全体成员通用职责

所有成员必须：

1. 熟悉最近逃生路线；
2. 熟悉水密门手动关闭方式；
3. 每日确认个人呼吸面罩位置；
4. 不隐瞒伤病、失眠、严重焦虑或幻听感；
5. 不擅自关闭传感器；
6. 不单独进入高风险舱段；
7. 不在通讯室、动力舱、医疗舱存放私人物品。

作息可因外部作业、月窗期、紧急维修调整。

## 4.2 巡检制度

**每日巡检项目：**

- 氧分压；
- 二氧化碳吸附效率；
- 湿度；
- 舱门密封；
- 水循环压力；
- 动力输出；
- 通讯缓存；
- 餐舱与医疗舱温度；
- 观测穹顶内外层状态。

**每周巡检项目：**

- 外壳应变趋势；
- 反应堆热交换效率；
- 蓄能阵列衰减；
- 水密门手动轮；
- 应急灯与呼吸面罩；
- 逃生舱自检；
- 深潜器接口密封圈。

## 4.3 舱门与水密隔离

水密门不得长期处于半开状态。红灯表示闭锁或故障，黄灯表示维护，绿灯表示可通行。

**禁止行为：**

- 用工具、箱体、管线卡住水密门；
- 未确认对侧压力时强行开启；
- 拆除门框传感器；
- 将个人权限卡借给他人；
- 在门缝附近堆放软包、衣物、纸箱。

## 4.4 氧耗、湿度与温控管理

站体氧耗受人员活动、情绪状态、温度、设备运行影响。
氧耗曲线出现周期性波动时，应首先检查：

1. 通风循环程序；
2. 二氧化碳吸附塔切换周期；
3. 热交换泵工作状态；
4. 人员活动记录；
5. 舱段湿度变化；
6. 外壳潮汐应力记录。

**标准氧耗记录间隔：** 每 6 小时一次。
**异常氧耗记录间隔：** 每 30 分钟一次，直至恢复稳定。

## 4.5 噪声与振动记录

深海站会受到潮汐、地层微动、热盐流及设备运转影响。
低频振动并不一定表示结构故障，但必须记录频率、持续时间和对应舱段。

**常见来源：**

- 热交换泵；
- 空气循环机；
- 水循环阀；
- 外壳热胀冷缩；
- 对接环残余应力；
- 海床微震；
- 远距离声学通讯回波。

若振动伴随灯光压降、舱壁结露或门锁误报，应升级为工程事件。

## 4.6 私人物品与公共空间

长期驻站允许携带有限私人物品，但应遵守：

1. 不携带易燃品；
2. 不携带强磁物；
3. 不携带挥发性香精或高敏过敏源；
4. 照片、书籍、音乐存储设备应登记；
5. 私人物品不得进入动力舱、通讯室主控台或医疗无菌柜；
6. 个人纪念物不得影响他人休息与工作。

公共空间应保持可清洁、可通行、可紧急隔离。

## 4.7 动物陪伴项目管理

子午线站参与 POU 封闭环境心理稳定项目，可在批准条件下配置一只小型陪伴动物。

**管理要求：**

- 动物须有独立医疗盒、保温箱、定位项圈；
- 食品、饮水、排泄物单独管理；
- 动物不得进入动力舱、通讯室、医疗无菌区和储备舱深处；
- 若动物出现拒食、持续躲藏、攻击性增强，应由医疗官记录；
- 紧急撤离时，动物保温箱应随最近人员携带，前提是不影响人员生命安全。

# 第五章　通讯规程

## 5.1 通讯限制说明

子午线站位于复杂热盐噪声带，常规高速通讯不可用。
对外通讯依赖低频声学链路及周期性稳定窗口。

通讯受以下因素影响：

- 热盐层扰动；
- 海床反射；
- 低频背景噪声；
- 站体姿态微偏移；
- 月相引起的声道稳定变化。

## 5.2 月窗期定义

“月窗期”指每约 29.5 天出现一次的低噪声通讯窗口。
窗口平均持续 **11 分钟**，实际可用时间可能缩短。

**月窗前准备：**

1. 提前 24 小时完成月报初稿；
2. 提前 12 小时压缩附件；
3. 提前 2 小时完成校验；
4. 提前 30 分钟冻结上传队列；
5. 月窗开启前 10 分钟通讯员值守；
6. 月窗结束后保存上传回执。

## 5.3 月报格式

标准月报包括：

1. 人员状态；
2. 设备状态；
3. 医疗摘要；
4. 氧耗与水耗统计；
5. 库存统计；
6. 科学数据摘要；
7. 异常事件；
8. 下月需求。

月报应简洁、可校验、可压缩。不得在月报中使用未登记缩写。

## 5.4 上传队列管理

上传队列由通讯员负责，站长复核。
所有附件必须具有：

- 文件编号；
- 来源舱段；
- 生成时间；
- 校验值；
- 压缩等级；
- 优先级。

优先级顺序：

1. 生命安全信息；
2. 站体结构异常；
3. 动力与生命保障状态；
4. 医疗情况；
5. 科学数据；
6. 库存与补给申请；
7. 个人短讯。

## 5.5 通讯室值守规范

通讯室值守期间：

- 不得离岗；
- 不得饮食；
- 不得接入私人终端；
- 不得关闭自动校验；
- 不得覆盖未上传缓存；
- 所有中断必须记录具体时间。

若通讯员失能，由站长指定替补。

## 5.6 离线缓存与故障处理

通讯中断并不表示数据丢失。所有待发送数据会进入离线缓存。

**故障处理顺序：**

1. 检查本地供电；
2. 检查终端温度；
3. 检查声学链路；
4. 检查压缩包校验值；
5. 切换备用缓存；
6. 记录故障码；
7. 通知站长。

# 第六章　工程维护

## 6.1 生命保障系统

生命保障系统包括：

- 氧气储备与生成；
- 二氧化碳吸附；
- 空气循环；
- 湿度控制；
- 微粒过滤；
- 气味与挥发物处理。

**警戒指标：**

| 项目 | 正常范围 | 警戒 |
|---|---|---|
| 氧分压 | 19.5%-23.5% 等效 | 低于 19% 或高于 24% |
| 二氧化碳 | ≤ 0.5% | 高于 0.8% |
| 相对湿度 | 35%-60% | 高于 70% |
| 舱内温度 | 18-24℃ | 低于 16℃或高于 27℃ |

## 6.2 动力系统

动力系统禁止非授权调整。
反应堆与蓄能阵列之间设有自动负载平衡系统。

**每日检查：**

- 输出功率；
- 冷却回路；
- 热交换效率；
- 母线温度；
- 备用电池容量；
- 应急照明供电。

## 6.3 水循环与废弃物处理

水循环系统分为饮用水、灰水、技术用水三类。
不得将样本液体、化学试剂或动物清洁废液倒入饮用水回收口。

## 6.4 热管理系统

热管理系统用于维持舱内舒适温度并排出动力舱废热。
若热风口温度异常，应检查：

1. 热交换阀；
2. 管道结露；
3. 风机转速；
4. 滤网堵塞；
5. 舱段门是否长期关闭。

## 6.5 外壳压力监测

外壳微形变在设计范围内属正常现象。
应关注趋势变化，而非单点读数。

**必须上报的情况：**

- 同一区域连续三次读数升高；
- 形变伴随渗水；
- 密封圈温度异常；
- 穹顶夹层湿度升高；
- 对接环无法完成闭锁。

## 6.6 观测穹顶维护

每日检查：

- 内层划痕；
- 夹层湿度；
- 除雾线路；
- 边缘密封圈；
- 外部灯阵。

禁止使用酒精、高腐蚀溶剂或粗糙布料擦拭穹顶内层。

## 6.7 深潜器对接维护

深潜器停靠后应完成：

1. 对接环盐雾清洁；
2. 外壳目视检查；
3. 推进器状态记录；
4. 电池与燃料读数；
5. 机械臂接口复位；
6. 样本舱密封确认。

# 第七章　医疗与心理稳定

## 7.1 入站适应期

新成员抵达后前 72 小时为适应期。常见反应包括：

- 失眠；
- 食欲下降；
- 轻度头痛；
- 耳压不适；
- 情绪敏感；
- 时间感减弱；
- 对舱壁噪声过度关注。

上述反应通常会在一至两周内缓解。若持续加重，应联系医疗官。

## 7.2 睡眠监测

长期驻站必须保持睡眠记录。
夜间值班人员应避免连续两晚值守。

睡眠异常包括：

- 连续三晚入睡困难；
- 频繁惊醒；
- 梦游；
- 睡眠中呼吸暂停；
- 醒后无法确认时间；
- 白天出现微睡眠。

## 7.3 氧分压与认知状态

氧分压异常、二氧化碳累积及睡眠不足均可造成判断力下降。
成员若出现明显迟钝、易怒、反应过激或语言混乱，应暂停其高风险工作。

## 7.4 封闭环境心理反应

深海封闭站可能引发：

- 孤独感；
- 空间压迫；
- 对重复声音敏感；
- 对私人记忆过度沉溺；
- 团队冲突；
- 对外部通讯的过度期待。

建议成员保持规律作息、适量运动、参与公共餐会，并定期与医疗官沟通。

## 7.5 高压深海驻站注意事项

请牢记：

1. 站体外部环境不可直接生存；
2. 不得因好奇擅自打开外部接口；
3. 深潜器外出必须两人以上审批；
4. 长时间凝视外部黑暗环境可能导致视觉疲劳与空间错觉；
5. 遇到无法解释的身体不适，优先排查缺氧、二氧化碳、睡眠不足和脱水。

## 7.6 心理稳定项目

子午线站允许使用以下心理稳定手段：

- 个人音乐库；
- 纸质书籍；
- 定期短讯；
- 运动计划；
- 陪伴动物；
- 团体餐会；
- 医疗官谈话。

任何成员不得嘲笑、干扰或破坏他人的合规心理稳定方式。

# 第八章　交接与轮换

## 8.1 轮换周期

子午线站标准驻站周期为 36 个月。
交接期建议不少于 72 小时，期间新旧两批成员共同驻站。

## 8.2 接收确认流程

新成员抵达后，按以下流程报到：

1. 站长确认身份；
2. 通讯员登记权限；
3. 医疗官完成初检；
4. 工程师说明当前设备状态；
5. 物资员交接库存；
6. 新成员签署接收确认。

**报到信息包括：**

- 姓名；
- 职务；
- 权限等级；
- 紧急联系人；
- 医疗禁忌；
- 个人设备清单。

## 8.3 设备交接

必须交接：

- 动力系统状态；
- 生命保障趋势；
- 水循环滤芯寿命；
- 外壳应变趋势；
- 通讯缓存；
- 深潜器状态；
- 逃生舱自检记录；
- 工具与备件库存。

## 8.4 库存交接

库存交接应逐项核对，不得只交接总数。

重点项目：

- 食品；
- 饮用水耗材；
- 医疗用品；
- 二氧化碳吸附剂；
- 密封圈；
- 电池；
- 工具；
- 动物用品。

## 8.5 医疗与心理档案交接

医疗档案由医疗官对医疗官交接。
心理记录仅提交摘要，不公开私人谈话细节，除非涉及安全风险。

## 8.6 离站前清理要求

离站成员须：

1. 清空个人柜；
2. 删除个人娱乐文件或转交本人存储器；
3. 归还权限卡；
4. 交回工具；
5. 完成医疗离站检查；
6. 签署离站确认。

# 第九章　应急处置

## 9.1 火灾

1. 就近拉下舱段隔离；
2. 使用对应类型灭火器；
3. 关闭局部通风；
4. 戴上呼吸面罩；
5. 通知站长；
6. 火灾后检测有毒气体。

## 9.2 进水

1. 确认进水舱段；
2. 关闭相邻水密门；
3. 切断非必要电力；
4. 启动排水泵；
5. 工程师评估破口；
6. 必要时放弃舱段。

## 9.3 供氧异常

1. 降低活动强度；
2. 集合至指定舱段；
3. 开启备用氧；
4. 检查二氧化碳吸附塔；
5. 医疗官评估人员状态；
6. 若无法恢复，准备撤离。

## 9.4 通讯中断

通讯中断时不得重复无序发送。
应保存数据，等待下一次窗口或使用压缩短报。

## 9.5 动力降级

1. 切换低功耗模式；
2. 保留生命保障、医疗、通讯、舱门控制；
3. 关闭非必要灯阵与娱乐系统；
4. 工程师检查蓄能阵列；
5. 站长决定是否准备撤离。

## 9.6 人员失能

1. 确认现场安全；
2. 通知医疗官；
3. 不单独搬运重伤人员；
4. 记录发现时间与地点；
5. 检查氧气、二氧化碳、温度；
6. 必要时进入医疗隔离流程。

## 9.7 强制撤离

强制撤离由站长宣布。
若站长失能，由剩余成员中权限最高者宣布。

撤离优先级：

1. 人员；
2. 医疗包；
3. 通讯定位信标；
4. 关键数据存储；
5. 陪伴动物保温箱；
6. 个人物品仅限随身件。
